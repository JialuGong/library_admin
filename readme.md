### how to run
######addtional remarks
front-end port:8001
back-end portt:8080
（请注意这两个端口有无被占用）
###### front-end
1. install node(推荐使用nvm)
	- windows 安装[nvm](https://github.com/coreybutler/nvm-windows/releases)
	- 安装node(v12.10.0)
	```shell
    nvm install 12.10.0
    ```
    - 使用node(v12.10.0)
    ```shell
    nvm use 12.10.0
    ```
2. run the front-end project
	进入front-end项目根目录：/front-end/mandarin_admin
    ```shell
    npm install
    npm run serve:randy
    ```
3. run the mockserver
	进入mockserver项目根目录：/mock
    ```shell
    npm install
    node apis.js
    ```
